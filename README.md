# my-search

## 说明
* 旨在提供一个用户本地的`文档（笔记）搜索`，用于用户便捷的搜索到自己的历史文档，以便于更高效的开发

## 环境
* laravel 5.4
* php 7.1
* elastic search 5.5.1 搜索引擎

## 安装
* 安装前端所需的包 `npm install`
* 安装php的一些包 `composer install`
* 如果需要二次开发，建议在`.gitignore`文件中添加忽略以下文件
> /resources/assets
  /public/fonts
  /public/js
  /public/css
